## 🔴 HIGH Priority (핵심 기능)

### 1. 하루 회고 (Daily Review) 기능
**설명**: 하루를 돌아보고 성찰하는 기능
**백엔드**: 이미 구현됨 (`src/routes/reviews.ts`)
**필요 작업**:
- [ ] 프론트엔드 UI 구현
- [ ] 회고 작성/수정 모달
- [ ] 회고 내용 표시 (접기/펼치기)
- [ ] 날짜별 회고 조회

**API 엔드포인트**:
```
POST   /api/reviews          # 회고 생성/수정
GET    /api/reviews/:date    # 특정 날짜 회고 조회
```

**UI 위치**: 메인 페이지 하단 (통계 카드 아래)

**예상 작업 시간**: 4-6시간

---

### 2. 주간 목표 (Weekly Goals) 관리
**설명**: 주간 목표 3개 설정 및 진행률 추적
**백엔드**: 이미 구현됨 (`src/routes/goals.ts`)
**필요 작업**:
- [ ] 주간 목표 설정 UI
- [ ] 진행률 슬라이더 (0-100%)
- [ ] 목표 완료/미완료 표시
- [ ] 주간 목표 카드 디자인
- [ ] 주차 자동 계산

**API 엔드포인트**:
```
POST   /api/weekly-goals              # 목표 생성
GET    /api/weekly-goals/current      # 현재 주 목표 조회
PATCH  /api/weekly-goals/:goalId/progress  # 진행률 업데이트
DELETE /api/weekly-goals/:goalId      # 목표 삭제
```

**UI 위치**: 별도 탭 또는 사이드바

**예상 작업 시간**: 6-8시간
